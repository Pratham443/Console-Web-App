Project On Console

To set up Visual Studio got to code.visualstudio.com and click on download in the top right corner. It will take you to another page. Click on your operating system and it will download it.

In the console screen you can see some messages from the computer. You can use it to see new messages the computer sends. To open the console screen right-click anywhere and click on inspect. if you drag the vertical bar it will show console. if you click on it it will show the messages sent. to print anything on the console screen use console.log("What you want to type") in javascript.
